Compatibility

Argus has been tested with:

  • perl 5.00503
  • perl 5.6.0 - but see perl issues
  • perl 5.6.1
  • perl 5.8.0 - but see perl issues
  • perl 5.8.1 - but see perl issues
  • perl ...
and
  • NetBSD
  • FreeBSD
  • Linux
  • Solaris
  • Mac OS X

And Argus should work with any OS that looks vaguely UNIX-like.

Argus has also been confirmed to not work with:

  • perl versions 1 - 4
  • microsoft windows
