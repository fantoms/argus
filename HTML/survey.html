Because sometimes I wonder...

how the software is used, statistics on numbers, etc. And you can help make the next version better. Once you have argus installed, configured, and running for a while, run the command
    argusctl status [1]
and mail the result to
    argus+survey @ tcp4me.com

Feel free to send it anonymously, or include comments, praise, death threats, etc. thanks

[1] - this command only exists in version 3.2 and later.
