Argus Demo

The following is a live system, actively monitoring roughly 1000 items.

While the author recommends installing on a nice quick modern system, for demonstration and comparison purposes, this demo is installed on an old, slow, single processor Sparc 20 (125 MHz, 128M RAM) a virtual server of unknown type. [1].

To demonstrate the user-views, and access-control features, there are several users set up, each with different permissions, and/or different views.

  • Read-Only Full View
  • Limited Read/Write Access, with Restricted View
  • Full Access to everything
  • Foreign Language View

[1] - Except for the several days following a freshmeat announcement, when the load on this poor little server goes through the roof.
